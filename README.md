# Contexto
Contexto em construção

## Técnologias usadas

Front-end:
> Desenvolvido usando: React, CSS3, HTML5, ES6

## Instalando Dependências
> Frontend
```bash
cd src/
npm install
``` 
## Executando aplicação
* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```

## Executando Testes

* Para rodar todos os testes:

  ```
    npm test
  ```
